1. Download latest build here
1b. To enable logging, in command line run
mkdir loggs
2. In command line run:
java -jar tango-0.1.5-SNAPSHOT-standalone.jar 1337
3. Browse to http://localhost:1337
4. Import a comptetition from a dance-perfect xml file
5. Select the comptetition and wait. (If its a big compttetition it might take a while)
6. Go to time scedule.
7. Open a second browser and go to http://localhost:1337/adjudicator
8. Place the browsers side by side.
9. Select a line in the scedule, and watch the adjudicator UI being updated.
